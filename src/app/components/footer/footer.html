Esta Política explica cómo recopilamos, usamos, protegemos y compartimos su información personal, cumpliendo con la Ley Orgánica de Protección de Datos Personales (LOPDP) de Ecuador. 1. Datos que Recopilamos Para prestar el servicio de internet y gestión de red, tratamos: • Datos de Registro: Nombre, número de cédula/RUC, dirección domiciliaria para instalación y contacto telefónico/email. • Datos de Red (ISP): Dirección IP (estática o dinámica), registros de tiempo de conexión, volúmenes de datos consumidos y metadatos de tráfico necesarios para la gestión técnica mediante plataformas como Wisphub. • Datos de Facturación: Historial de pagos, datos de tarjetas o cuentas bancarias gestionados a través de nuestros sistemas de facturación de terceros. 3. Finalidades y Bases Legales Tratamos sus datos basándonos en las siguientes condiciones legales: 1. Ejecución del Contrato: Para instalar el servicio, brindar soporte técnico y gestionar la cobranza. 2. Obligación Legal: Para conservar registros de tráfico por el periodo exigido por la Ley de Comercio Electrónico y regulaciones de ARCOTEL (fines de auxilio a la justicia). 3. Interés Legítimo: Para garantizar la ciberseguridad de nuestra infraestructura y prevenir fraudes en la red. 4. Consentimiento: Únicamente si usted lo autoriza, para envío de ofertas comerciales o promociones. 4. Uso de Plataformas de Terceros (Encargados) Para una gestión eficiente, WIIFIBER utiliza proveedores especializados: • Gestión de Red: Utilizamos Wisphub para el aprovisionamiento y control de tráfico. • Sistemas Contables: Los datos de facturación se procesan en sistemas externos que cumplen con estándares de seguridad y cifrado. WIIFIBER ha firmado acuerdos de confidencialidad con estos proveedores para asegurar que sus datos no sean usados para fines distintos a los aquí descritos. 5. Transferencias Internacionales En caso de que nuestros proveedores de software utilicen servidores fuera de Ecuador (ej. nubes de AWS o Google), WIIFIBER asegura que dichos proveedores cumplan con niveles adecuados de protección de datos según los estándares internacionales aceptados por la autoridad ecuatoriana. 6. Conservación de los Datos • Datos de contrato y facturación: Se conservarán por 7 años (según el Código Tributario). • Registros de tráfico (Logs): Se conservarán por el plazo legal de 7 años para requerimientos judiciales. • Datos de marketing: Se conservarán hasta que usted revoque su consentimiento. 7. Sus Derechos (ARCO+) Usted tiene derecho a: • Acceder a sus datos personales. • Rectificar información inexacta. • Eliminar sus datos (cuando no exista una obligación legal de mantenerlos). • Oponerse al uso de sus datos para fines comerciales. • Portabilidad de sus datos técnicos. Para ejercer estos derechos, descargue nuestro Formulario de Solicitud en la web y envíelo a protecciondedatos@wifiber.com.ec. 8. Seguridad de la Información WIFIBER implementa medidas técnicas de vanguardia, incluyendo cifrado de datos, protocolos de seguridad en la nube y acceso restringido basado en roles para sus empleados, minimizando el riesgo de filtraciones. 9. Cambios en la Política WIIFIBER se reserva el derecho de actualizar esta política. Cualquier cambio sustancial será notificado a través de nuestra página web o por correo electrónico. .
